# Antenna Laboratory Setup

## Introduction
This experiment focuses on studying different antenna types and their behavior in a controlled laboratory environment. Using an antenna trainer system and several antenna elements, the experiment allows observation of radiation patterns, antenna alignment, and signal reception characteristics. The setup provides practical experience with antenna positioning equipment and common antenna structures used in communication systems.

## Objectives
- Understand the basic operation of different antenna types.
- Observe how antenna orientation affects signal transmission and reception.
- Identify the components used in an antenna measurement system.
- Familiarize with the antenna trainer and positioning equipment.
- Study the characteristics of monopole, dipole, and directional antennas.

## Equipment

2. Linear Antenna Measurement Scale
3. Antenna Trainer Unit
